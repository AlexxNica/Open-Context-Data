Open Context
Web-based research data publication
This represents an early stage experiment in using GitHub to host Open Context datasets.

Open Context publishes archaeological data (mainly). The purpose of this repository is to share Open Context's data in a way that is easy for anyone to reuse, expand upon, and modify, all in the context of powerful version control features.

Dependencies

The XML files are pretty straightforward, using elements of the "Archaeological Markup Language" (ArchaeoML) from the University of Chicago OCHRE project. There are some additional Open Context specific namespace elements (no recent XSD yet, that will come!).
