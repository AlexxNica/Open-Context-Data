Web-based research data publication

About Open Context Data on GitHub

This represents an early stage experiment in using GitHub to host Open Context datasets. The main point of this is to add more transparency to versioning of Open Context data, and eventually, facilitate more collaboration by making it easier to "fork" these data for new applications.

Open Context publishes archaeological data (mainly). These data are stored as XML documents which have been added to this GitHub hosted repository. Please note, Open Context does not rely upon GitHub for long-term archiving and data preservation. Open Context also works with the California Digital Library's Merritt repository. Open Context uses of GitHub mainly to encourage collaboration and transparency, and not for data preservation.

Explore and Download Open Context Data:

  • Browse the data in GitHub: ekansa/Open-Context-Data
  • Download all data (Zip-compressed): zip
  • Download all data (Tar-format): tar
  • You can also clone the project with Git by running:
    $ git clone git://github.com/ekansa/Open-Context-Data


Notes about Open Context Data on GitHub

The XML files are pretty straightforward, using elements of the "Archaeological Markup Language" (ArchaeoML) from the University of Chicago OCHRE project. There are some additional Open Context specific namespace elements (no recent XSD yet, that will come!). To use these data, please note the following:

  • Licenses Each XML document has Creative Commons licensing data in each document. Most carry the Creative Commons Attribution license, but some of the earlier datasets have the (dreaded, less-than-ideal) non-commercial restriction.
  • Authorship / Attribution Each XML document has Dublin-Core metadata noting "creators" and "contributors". Please cite and attribute these individuals in all works based on these data. Please also use the Dublin Core identifier element (an Open Context URL/URI) when citing these data.
  • More Information A brief introduction to the concepts behind these XML data can be found at Open Context's "About" pages. Some of the XML documents have an early stab at describing Linked Data relations. These are introduced here.
